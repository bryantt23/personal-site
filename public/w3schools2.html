Dropdowns

Use the Bootstrap dropdown class to indicate a dropdown menu.

To actually build the dropdown menu use the dropdown-menu class.

Use a button or a link with the class dropdown-toggle and also set data-toggle equal to "dropdown" in order to open the dropdown menu.

Notice the divider between the JavaScript and SQL and also the use of a Dropdown Header.

Dynamic Pills

Clicking on the pill changes the pane's content.

To make the pill toggable, add the data-toggle="pill" attribute to each link.

Next create a tab-pane class that has a unique ID for each tab. Wrap all of these tabs inside a div element with the class name tab-content.
